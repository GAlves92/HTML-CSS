Summary: Latitude vs. X




The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to gather dataset on over 500 cities around the world.

After gathering the data, we used Matplotlib to plot different aspects of the weather vs. latitude. Aspects such as temperature, cloudiness, wind speed and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations discovered.
